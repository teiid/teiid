
This exaple will demonstrate the following: 

	* How to use JDBC Translator extract the data in a RMDBS
	
	* How to define a view using DDL
	
	* How to define a translator override to support native queries
	
	* How to define a materialized VIEW using DDL and load the external materialized table
	
This example uses the H2 database, which is referenced as the "accounts-ds" data source, but the creation SQL can be adapted to another database if you choose.
